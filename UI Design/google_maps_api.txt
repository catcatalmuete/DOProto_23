We could use google maps services to allow users to send pickup locations and meeting places via google maps integration.

This would be added after we have most of our webpage done, using an HTTP request

The process involves generating an API key, creating an HTML container for the map, add external script by google inside 
HTML doc, and then bring the map inside HTML container using JavaScript (see geeksforgeeks example below)

More details at: https://developers.google.com/maps/documentation/embed/get-started

Further Resources: https://support.google.com/maps/answer/144361?hl=en&co=GENIE.Platform%3DDesktop

Tutorial & Example: https://www.geeksforgeeks.org/how-to-add-google-maps-to-a-website/
